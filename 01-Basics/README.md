# Basics

This section introduces the fundamentals of Bash scripting, including:

- Basic commands
- Script structure
- Running scripts

## Examples

1. **Hello World**:
   ```bash
   #!/bin/bash
   echo "Hello, World!"
   ```

2. **Basic Arithmetic**:
   ```bash
   #!/bin/bash
   echo $((2 + 2))
   ```

## Exercises

- Write a script that prints your name.
- Write a script that calculates the sum of two numbers.
